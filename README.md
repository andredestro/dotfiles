# andre's dotfiles

> Config files and installs some apps

## Installation

### Dependencies

First, make sure you have all those things installed:

- `git`: to clone the repo
- `curl`: to download some stuff

### Install

Then, run these:

```sh
$ git clone https://github.com/andredestro/dotfiles.git ~/.dotfiles
$ cd ~/.dotfiles
$ ./script/bootstrap
```
